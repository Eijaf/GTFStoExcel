Prérequis (à vérifier à la première utilisation uniquement): 
  L'ordinateur doit avoir python d'installer (cocher toute les options à l'instalation).
  La bibliothèque Openpyxl non installée par défault doit y être présente. Pour ce faire : win + R > cmd > py > pip install openpyxl
  
Utilisation :
 (Il est préférable d'utiliser un GTFS sur une semaine type)
 Créer un nouveau dossier et y importer les fichiers GTFS suivant : stop_time.txt, stops.txt, trips.txt, calendar.txt et routes.txt
 Si les villes sont souhaitées, ajouter le fichier city.txt à créer manuellement.
 Copier/coller le programme dans ce même dossier.
 Exécuter le programme.
 Une console apparait avec des informations ; un fichier .bd et un fichier .xls sont créer si le déroulement s'effectue corectement.
 Le fichier .xls contient les informations du GTFS, elle doivent êtres retoucher selon les consigne du document "Mise en Page"
 
ATTENTION : si le programme doit être relancer, effacer les fichier .db et .xls. En cas d'oublie, le programme ne s'executera pas correctement.
